# 📰 Fake News Verification Web App

A simple and interactive web application that helps users identify whether a piece of news is **Real or Fake** using Machine Learning techniques.

🔗 Live Website  
https://verify-fake.vercel.app/

---

## 📌 Project Overview

Fake news spreads quickly on the internet and social media. This project aims to reduce misinformation by allowing users to quickly check whether a news article or statement is likely **real or fake**.

Users simply enter a news headline or text, and the system analyzes the content and predicts the result.

---

## 🚀 Features

- 🔍 Detects whether news is **Real or Fake**
- 🧠 Uses **Machine Learning model for prediction**
- 💬 Simple and user-friendly interface
- ⚡ Fast response time
- 🌐 Accessible from any device through the web

## ⚙️ How It Works

1. User enters a news headline or article text.
2. The input text is cleaned and processed.
3. The processed text is converted into numerical features using **TF-IDF Vectorization**.
4. The trained machine learning model analyzes the features.
5. The system predicts whether the news is **Real or Fake**.
